Tắt Schema tự động của Rank Math cho Trang chủ

⚠️
Tại sao cần tắt? Khi bạn paste schema custom vào WPCode, Rank Math vẫn tự động gen schema của nó → trang chủ bị duplicate 2 schema cluster → Google confused, có thể ignore cả hai. Cần tắt Rank Math để schema custom của bạn là nguồn duy nhất.

📋 Quy trình 3 bước

1 Cài plugin WPCode (nếu chưa có)
  1. Vào WordPress Admin → Plugins → Add New
  2. Tìm: WPCode hoặc Insert Headers and Footers Code Snippets
  3. Click Install Now → Activate
  4. Sau khi active, menu Code Snippets sẽ hiện trong sidebar admin
2 Tạo PHP Snippet mới trong WPCode
  1. WordPress Admin → Code Snippets → + Add Snippet
  2. Chọn "Add Your Custom Code (New Snippet)"
  3. Khi popup chọn loaij code hiện lên, chọn PHP Snippet (không phải HTML, không phải JavaScript)
  4. Title: "Disable Rank Math schema on homepage"
  5. Code Preview: copy & paste code PHP bên dưới
  6. Insertion:
    • Insertion Method: Auto Insert
    • Location: Run Everywhere
  7. Bật toggle Active (góc trên phải)
  8. Click Save Snippet
3 Clear cache + Verify
  1. Nếu có WP Rocket: Settings → Clear All Cache
  2. Nếu có Cloudflare: Caching → Configuration → Purge Everything
  3. Caching plugin khác (W3 Total Cache, WP Super Cache...): clear tương ứng
  4. Test: mở View Source (Ctrl+U) trang chủ → tìm rank-math-schema → phải KHÔNG còn thấy (hoặc chỉ còn 1 block từ WPCode của bạn)
  5. Test: Google Rich Results Test → paste URL trang chủ → phải hiển thị đúng 5 entity sạch (WebPage, ImageObject, BreadcrumbList, WebSite, Organization)

❓ FAQ


      

        Làm sao biết Rank Math đã bị tắt chưa?
        
Mở View Source trang chủ (Ctrl+U), tìm chuỗi rank-math-schema. Nếu KHÔNG còn thấy → đã tắt thành công. Nếu vẫn thấy → chưa clear cache hoặc snippet WPCode chưa Active.

      


      

        Tin này có ảnh hưởng đến schema của các page khác (Article, Product) không?
        
KHÔNG. Function chỉ tác động đến trang chủ (is_front_page + is_home). Các trang Article, Product, Category... vẫn được Rank Math gen schema bình thường.

      


      

        Tại sao không paste trực tiếp vào functions.php của theme?
        
Có thể paste vào functions.php của child theme nếu bạn có. Nhưng KHÔNG nên paste vào parent theme — mỗi lần update theme sẽ mất code. WPCode an toàn hơn vì tách rời với theme.

      


      

        Nếu muốn KHÔI PHỤC schema Rank Math thì sao?
        
Vào WPCode → tìm snippet "Disable Rank Math schema on homepage" → tắt toggle Active. Rank Math sẽ tự động gen lại schema cho trang chủ.

      


      

        Dùng Yoast SEO thay vì Rank Math thì sao?
        
Code này CHỈ hoạt động với Rank Math. Để tắt schema Yoast trên trang chủ, dùng code khác:

        
add_filter( 'wpseo_json_ld_output', function( $data ) {
    if ( is_front_page() || is_home() ) {
        return array();
    }
    return $data;
}, 99 );

🛡️ Brand Protection mode dùng khi nào?

Khi brand của bạn thuộc ngành không thể đăng ký pháp lý tại VN (gambling, crypto, adult, grey market). Mode này sẽ:

  • Cảnh báo keyword cấm/illegal theo luật VN (vd: "cá cược", "casino", "đá gà", "đổi thưởng", "nổ hũ", "xổ số")
  • Gợi ý từ ngữ "reframe" an toàn (thay vì "casino" → "nền tảng giải trí số", thay vì "đổi thưởng" → "sản phẩm số")
  • Loại bỏ các field cần pháp lý (taxID, address chính thức)
  • Tập trung vào sameAs + alternateName + foundingDate + areaServed Vietnam

🇻🇳 Tối ưu riêng cho thị trường Việt Nam (phiên bản v5)

Phiên bản này được tối ưu riêng cho website phục vụ user Việt Nam:

  • inLanguage = "vi" cố định (không cho đổi để tránh nhầm)
  • knowsLanguage = ["vi"] mặc định
  • areaServed mặc định chỉ Vietnam (1 quốc gia)
  • Default sameAs ưu tiên social VN phổ biến: Facebook (rất quan trọng tại VN), Zalo, Telegram, TikTok, YouTube
  • Default contact: email + Telegram support (Telegram được dùng phổ biến trong ngành VN)
  • Banned keyword mở rộng đặc thù VN: nổ hũ, bắn cá, xổ số, số đề, kèo bóng, soi cầu, đổi thưởng, hoàn trả 100%, thưởng nạp đầu...
  • Detect VN Regulator triggers (cụm từ Bộ Công an + Bộ TT-TT phát hiện web cờ bạc)

⚠️ Các keyword cấm/rủi ro tôi check

Tool sẽ tô VÀNG/ĐỎ các field chứa (đặc biệt cho thị trường Việt Nam):

  • Gambling/Cờ bạc: cá cược, casino, slot, đá gà, xóc đĩa, tài xỉu, lô đề, sòng bài, bookmaker, bet, gambling, jackpot, nổ hũ, bắn cá, xổ số, số đề, kèo bóng, soi cầu, đổi thưởng
  • Promo aggressive (nhà cái VN): "tặng XXXK", "code XYZ", "đăng ký nhận", "link truy cập", "hoàn trả 100%", "thưởng 200%"
  • VN Regulator triggers: "web cá cược", "game đỏ đen", "tip miễn phí" (các từ Bộ Công an + Bộ TT-TT thường dùng để phát hiện)
  • Spam indicator: emoji trong name/title, ALL CAPS, pipe character lạm dụng
  • Brand inconsistency: URL trong sameAs khác username brand
